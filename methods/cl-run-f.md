# cl-runF / clock.runFor

Run all pending timers and advance the clock by a given duration (ms).

## Syntax

| Surface | Syntax |
|---|---|
| CLI | `vibium clock run-for <ms>` |
| MCP | `page_clock_run_for { ms }` |
| JS | `page.clock.runFor(ms)` |
| Python | `page.clock.run_for(ms)` |
| Java | `page.clock.runFor(ms)` |
