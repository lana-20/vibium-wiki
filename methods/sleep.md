# sleep

Pause execution for a fixed duration (ms). Use as a last resort when no actionable signal exists.

## Syntax

| Surface | Syntax |
|---|---|
| CLI | `vibium sleep <ms>` |
| MCP | `browser_sleep { ms }` |
| JS | `page.sleep(ms)` |
| Python | `page.sleep(ms)` |
| Java | `page.sleep(ms)` |
