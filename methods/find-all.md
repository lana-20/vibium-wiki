# findAll

Find all elements matching a selector; returns a list of `@ref` handles.

## Syntax

| Surface | Syntax |
|---|---|
| CLI | `vibium find <selector> --all` |
| MCP | `browser_find_all { selector }` |
| JS | `page.findAll(selector)` |
| Python | `page.find_all(selector)` |
| Java | `page.findAll(selector)` |
